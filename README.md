============================================
     Bandwidth Calculator - Портативная версия
============================================

О ПРОГРАММЕ:
Bandwidth Calculator - это конвертер пропускной способности сети,
который позволяет переводить значения между различными единицами
измерения скорости передачи данных.

ВОЗМОЖНОСТИ:
- Конвертация между 12 единицами измерения
- Поддержка: bits/bytes, K, M, G, T, P
- Настраиваемая точность (1-10 знаков после запятой)
- Автоматическая конвертация по Enter
- Копирование отдельных или всех результатов
- Современный интерфейс Windows 11

ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ:
1. Запустите BandwidthCalculator.exe
2. Введите значение в любое поле
3. Нажмите Enter или кнопку "Конвертировать"
4. Все поля автоматически обновятся

ТРЕБОВАНИЯ К СИСТЕМЕ:
- Windows 10 или новее (рекомендуется Windows 11)
- 64-битная система (x64)
- Не требует установки .NET Framework
